# Tri-Valley Readers — Book Club Website

A simple, multi-page website for the Tri-Valley Readers book club.

## Pages

- **Home** (`index.html`) — Welcome, tagline, and overview of what the club does
- **Reading List** (`reading-list.html`) — Tabs for "Currently Reading" and "Past Reads" with sample books
- **Membership** (`membership.html`) — Who can join, what’s included, expectations, and contact info

## Tech

Plain HTML, CSS, and a small amount of JavaScript for the reading-list tabs. No frameworks or dependencies.
